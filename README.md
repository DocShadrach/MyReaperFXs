# MyReaperFXs

A collection of custom JSFX effects for the REAPER Digital Audio Workstation.

These effects are experimental tools focused on audio processing concepts, workflow ideas,
and sonic experimentation inside REAPER.

## Usage

You are allowed to:

- Use these JSFX effects inside REAPER
- Load, run and test them for personal, non-commercial purposes

## How to add this repository to ReaPack

1. Open **REAPER**
2. Go to **Extensions → ReaPack → Import a repository**
3. Paste the following URL:
[url]https://raw.githubusercontent.com/DocShadrach/MyReaperFXs/master/index.xml[/url]
4. Click **OK**
5. Go to **Extensions → ReaPack → Browse packages**
6. Find the desired JSFX and install it
7. Restart REAPER if required
